# Portfolio Website – HTML, CSS & JavaScript Assignment

📌 **Project Overview**  
This project is a personal portfolio website developed using HTML5, CSS3, and JavaScript.  
It demonstrates the use of basic web development concepts such as page structure, styling, layout design, responsiveness, and interactivity.

---

🧱 **Website Structure**  

The website includes the following sections:

1. **Header**  
   - Displays my name: Riya Gupta  
   - Includes a short tagline: Web Developer | Student | Learner

2. **Navigation Bar**  
   - Contains links to: Home, About, Projects, Contact  
   - Helps users navigate to different sections of the page

3. **Home Section**  
   - Describes my hobby: Coding  
   - Includes a paragraph explaining why I like coding  
   - Displays an ordered list showing three benefits of coding  
   - **JavaScript Interactivity:**  
     - Users can add new benefits using an input box and “Add Benefit” button  
     - Each list item has a “Delete” button to remove it dynamically  

4. **About Section**  
   - Provides a brief introduction about my interest in web development  
   - Mentions my learning experience with HTML and CSS

5. **Projects Section**  
   - Displays three sample projects:  
     - Project One: Simple HTML website  
     - Project Two: Portfolio webpage using CSS  
     - Project Three: Responsive landing page  
   - Projects are arranged using CSS Flexbox for proper alignment

6. **Contact Section**  
   - Includes a contact form with Name, Email, and Message fields  
   - **JavaScript Interactivity:**  
     - Displays an alert when the form is submitted successfully  

7. **Footer**  
   - Contains social media links (LinkedIn and GitHub)  
   - **JavaScript Interactivity:**  
     - Dynamically displays current date and time, updated every second  

---

🎨 **Styling & Design**  
- External CSS file used for styling  
- Google Font (Poppins) used for typography  
- Consistent colors and spacing applied across the website  
- Flexbox used in the Projects section  
- Hover effects applied on navigation links and buttons  

📱 **Responsiveness**  
- The website is responsive for smaller screen sizes  
- Media queries are used to display project cards in a column layout on mobile devices  

🗂 **Project Structure**  
- `index.html`  
- `style.css`  
- `main.js` (JavaScript interactivity)  
- `README.md`  
- `.gitignore`  

🚀 **Technologies Used**  
- HTML5  
- CSS3  
- Flexbox  
- Google Fonts
